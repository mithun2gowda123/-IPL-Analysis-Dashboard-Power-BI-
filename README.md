# -IPL-Analysis-Dashboard-Power-BI-


## Overview
This project is an interactive Power BI dashboard built to analyze Indian Premier League (IPL) data using visualization and analytics techniques.  
It provides insights into team performance, player statistics, match outcomes, and season trends.  
The dashboard helps identify top-performing teams, highest run scorers, leading wicket takers, and venue-based patterns.  
This project demonstrates data cleaning, modeling, DAX calculations, and dashboard development in Power BI.

## Objectives
- Analyze IPL team and player performance  
- Visualize match statistics and season trends  
- Compare teams based on wins, runs, and wickets  
- Build an interactive dashboard using Power BI  
- Generate insights from cricket data

## Dataset Used

### Matches Dataset
Contains match-level information:
- Match ID  
- Season  
- Teams  
- Venue  
- Toss winner  
- Match winner  
- Player of the Match  

Dataset: [matches(1).csv](./matches(1).csv)

### Deliveries Dataset
Contains ball-by-ball information:
- Match ID  
- Innings  
- Over and Ball  
- Batsman and Bowler  
- Runs scored  
- Wickets  
- Extras  

Dataset: [deliveries(1).csv](./deliveries(1).csv)
